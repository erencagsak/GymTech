<h1>GYM TECH</h1>

<h3>What is "Gym Tech" ?</h3>

<p>Gym Tech is a gym application for android phones. Users can register/login, show the gym movements and more...</p>

<h3>What did i use program/language for make this project?</h3>

- Android Studio
- Kotlin
- FireBase
